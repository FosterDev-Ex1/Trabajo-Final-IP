# Proyecto Final Introducción a la Programación / ICI

## Hospital Calixto García: Sistema de Monitoreo de Parámetros

Este repositorio contiene el código fuente para el trabajo final de la asignatura de Programación I / ICI, cuyo objetivo es gestionar y clasificar el riesgo de pacientes en un entorno hospitalario.

### Tecnologías Utilizadas
* **Lenguaje:** C++
* **Control de Versiones:** Git

### Problemáticas Resueltas (Incisos Clave)
* **Registro Dinámico:** Uso de `std::vector` para crear un "array hospital" de tamaño variable (Inciso B).
* **Clasificación de Riesgo:** Implementación de un subprograma (`determinarGravedad`) para clasificar pacientes según criterios médicos (Inciso E).
* **Análisis Estadístico:** Cálculo de promedios, máximos y porcentajes (Incisos F, G, H, I).
* **Aplicación Matemática:** Demostración de la Ortogonalidad Vectorial en los datos de síntomas (Inciso K).

## Utilidad Social del Código
El sistema permite al personal del hospital tomar decisiones rápidas sobre el triage de pacientes, priorizando a aquellos con clasificación "Grave" o "Crítico", contribuyendo a una gestión más eficiente de recursos médicos en situaciones de alta demanda.
